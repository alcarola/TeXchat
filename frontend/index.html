About TeXchat

TeXchat is a simple web chat service that renders TeX math.

The goal is to make it trivially easy to have a conversation with someone else in the web, using mathematical symbols. "0_j := 0_j - a^3 J(0) d0 - sum(phi_i)" just doesn't cut it, so instead, use \( \theta_j := \theta_j - \alpha^3 J (\theta) \partial \theta - \sum \phi_i \)

Currently, the prettiest math rendering is TeX math, which MathJax has ellegantly brought to the web. This is coupled with a simple chat service on top of socket.io.

If you have any suggestions, feedback, issues, etc: please start a topic in the github issues page. Feel free to contact the author directly, but github is preferable.
